## Setup

```bash
$ git clone https://github.com/2gbeh/fedora.git
$ cd fedora
$ npm cache clean --force
$ npm install
# OR npm install --legacy-peer-deps

TODO: Copy .env credentials

$ npx expo start -c
```

> Local: http://localhost:8081
