## Setup

```bash
$ (ssh) git clone git@github.com:Plateaumed/ehe-react-tab-phc.git

$ cd ehe-react-tab-phc

# $ npm cache clean --force
# $ npm install --legacy-peer-deps
$ npm install

# Copy .env credentials

$ npx expo start -c
```
