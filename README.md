## Setup

```bash
$ git clone https://github.com/2gbeh/fedora.git
# OR (ssh) git clone git@github.com:2gbeh/fedora.git

$ cd fedora

$ npm cache clean --force

$ npm install
# OR npm install --legacy-peer-deps

# OPTIONAL copy .env credentials

$ npx expo start -c
```
